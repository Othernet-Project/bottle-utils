AJAX (bottle_utils.ajax)¶

The bottle_utils.ajax module provides decorators for working with AJAX requests.

Decorators¶

bottle_utils.ajax.ajax_only(func)[source]¶

This decorator simply tests if request is_xhr and aborts any requests that are not XHR with an HTTP 400 status code. Keep in mind, though, that AJAX header (‘X-Requested-With’) can be faked, so don’t use this decorator as a security measure of any kind.

Example:

@ajax_only
def hidden_from_non_xhr():
    return "Foo!"
Parameters:func – request handler
Raises:bottle.HTTPResponse
Returns:wrapped function
